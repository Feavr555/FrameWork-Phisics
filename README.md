# FrameWork-Phisics

Requisitos para usar el programa, tener instalado:
make
gcc
Como comandos de terminal.

```bash
make
```
Y compila todo para linux.
```bash
make -f MakefileWIN
```
Compila todo para windows con el compilador cruzado, de linux para windows.
```bash
make -f MakefileWIN CC=gcc
```
Compila para windows desde windows.
